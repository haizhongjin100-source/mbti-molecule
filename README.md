# MBTI分子 — 化学反应预测器

> *"每个人都是一种独特的元素，当两种元素相遇，便产生一场化学反应。"*

选择任意两个 MBTI 人格，MBTI分子会生成一场专属的"人际化学反应"分析——从反应剧烈程度、沟通模式、信任建立，到典型场景和相处建议，全部以化学实验报告的形式呈现。

---

## ⚗️ 功能概览

| 功能 | 说明 |
|------|------|
| **化学反应预测** | 136 种人格组合的化学反应描述，支持友谊/恋爱/职场三维度切换 |
| **反应方程式** | 基于数据自动生成伪化学方程式（产物、催化剂、温度、反应热） |
| **最佳相处温度计** | match% 映射为温度读数，液柱动画直观展示契合程度 |
| **人格详情** | 16 种 MBTI 完整档案：特质、沟通方式、情感模式、职业建议、成长指南 |
| **组合全览** | 选择一个人格，查看与所有 15 种类型的化学反应摘要 |
| **快速测试** | 4 道趣味选择题，测出你的 MBTI 人格类型 |
| **CP 排行榜** | 基于匹配度对全部 136 组组合排名，支持按强度类型筛选 |
| **分子图谱** | Canvas 力导向关系网络图，可视化 16 种人格的关系网络 |
| **人格画像** | 输入名字生成专属人格标签卡，支持保存为图片 |
| **16型周期表** | MBTI 人格以化学元素周期表形式排列，按四大气质分组展示 |
| **宿舍兼容性测试** | 2-8 人宿舍分析：氛围判定、冲突预警、作息兼容、任务分配 |

## 🧪 页面结构

```
MBTI分子/
├── index.html          # 主页 — 化学反应预测器
├── personality.html    # 人格详情 — 16种人格完整档案
├── combos.html         # 组合全览 — 一个MBTI × 所有组合
├── quiz.html           # 快速测试 — 4题测出MBTI
├── ranking.html        # 最佳CP排行榜
├── molecule.html       # 分子图谱 — 关系网络可视化
├── periodic.html       # 16型周期表 — MBTI元素周期表
├── portrait.html       # 人格画像 — 生成专属标签卡
├── dorm.html           # 宿舍兼容性测试
└── scripts/
    └── data.js         # 共享数据（3×136组反应 + 人格档案 + 配置）
```

### 页面跳转

```
index.html（主页）
  ├─ 选择器中文名 → personality.html?type=XX
  ├─ 结果区 MBTI 代码 → personality.html?type=XX
  ├─ 人格鉴定 → quiz.html
  ├─ 组合全览 → combos.html
  ├─ CP排行榜 → ranking.html
  ├─ 分子图谱 → molecule.html
  ├─ 人格画像 → portrait.html
  └─ URL 参数 ?mbti1=XX&mbti2=YY 自动预选

personality.html → index.html（测试化学反应）→ combos.html → dorm.html
quiz.html → personality.html / combos.html / index.html
molecule.html → index.html（点击节点查看反应）
periodic.html → index.html / personality.html（点击元素卡片）
portrait.html → index.html / personality.html
dorm.html → 所有页面可达
```

## 🔬 数据说明

- **136 组化学反应**：覆盖全部 C(16,2) + 16 种同类型组合
- **三维度数据**：友谊版 / 恋爱版 / 职场版各 136 条，合计 408 条
- **16 种人格档案**：每种人格包含概述、特质、沟通、情感、职业、成长、摩擦、代表人物
- **数据字段**：intensity、title、desc、match、comm、trust、conflict、long、energy、scene、scenes、tips
